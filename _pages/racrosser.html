RACROSSER

RACROSSER is an integrated software that enables calculation of Radar Cross Section for objects of arbitrary shapes and materials. Software consists of:

  • preprocessor that allows to load mesh and specify input parameters;
  • multiprocessing computational core, based on the approach of physical optics complemented with physical theory of diffraction concepts;
  • postprocessor for rendering results.

Software is written in Python (with the use of Qt, OpenGL and Matplotlib). Some heavy computational routines are ported to C++.
