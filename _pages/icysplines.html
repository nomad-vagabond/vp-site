IcySplines

IcySplines is a software for data fitting with splines. It is inspired by such data acquisition and curve fitting tools as Gsys, CurveExpert, DataFit and a few others. IcySplines aims at integrating basic capabilities of the mentioned tools and makes an accent on the use of splines for curve fitting.

The software allows to extract data from images and tables and provides a set of tools for interpolation and approximation with spline functions and parametric spline curves. It is written in Python with the use of SciPy, PyQt and Matplotlib.

The project is in the development phase.
